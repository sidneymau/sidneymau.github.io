# Colophon

This website was built using [Tachyons](http://tachyons.io/) and is hosted by [GitHub Pages](https://pages.github.com/).

This website uses the [Fira Sans](https://bboxtype.com/typefaces/FiraSans/#!layout=specimen) typeface.

The CV was typeset with [XeLaTeX](https://tug.org/xetex/) using [STIX Fonts](https://www.stixfonts.org/).
